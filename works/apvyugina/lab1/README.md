# Отчет по лабораторной работе №1

## 1. Задачи лабораторной

Лабораторная работа направлена на изучение основ программирования на CUDA. Основные задачи:

- Изучение базовых концепций CUDA: kernel-функции, потоки, блоки
- Реализация параллельного алгоритма с использованием GPU
- Освоение работы с памятью GPU (выделение, копирование, освобождение)
- Практическое применение CUDA для решения задачи отрисовки квадрата

В данной работе реализована программа, которая с использованием CUDA рисует квадрат заданного размера из символов '*', где звездочками отмечены только границы квадрата.

## 2. Инструкция по запуску

**Требования:**
- NVIDIA GPU с поддержкой CUDA
- Установленный CUDA Toolkit
- Компилятор nvcc

**Сборка проекта:** `make build`.
Это создаст директорию `build/` и скомпилирует исполняемый файл `draw_function`. Для удаления скомпилированных файлов выполните команду `make clean`.

**Запуск программы:**

Программа принимает один аргумент командной строки - размер квадрата:

```bash
./build/draw_function <размер>
```

## 3. Примеры работы
```bash
./build/draw_function 3
```
```bash
***
* *
***
```

## 4. Выводы
В ходе выполнения лабораторной работы были освоены базовые принципы программирования на CUDA: создание kernel-функций, организация потоков и блоков, а также работа с памятью устройства. Практическое применение полученных знаний позволило реализовать параллельный алгоритм отрисовки квадрата на GPU, что демонстрирует возможность использования CUDA для задач, допускающих параллельное выполнение.
