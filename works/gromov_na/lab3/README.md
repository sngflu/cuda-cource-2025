# Lab 3: CUDA Sobel Operator

## Описание

Реализация оператора Собеля для обнаружения границ в изображениях на CUDA. Программа загружает изображение в формате PGM, применяет фильтр Собеля на GPU и сохраняет результат с выделенными границами.

## Требования

- CUDA Toolkit 11.0 или выше
- GPU с compute capability 7.0+
- Linux с GCC 9.0+
- Make

## Компиляция

```bash
make
```

## Запуск

```bash
./sobel [input.pgm] [output.pgm]

# Примеры:
./sobel bu.pgm edges.pgm
./sobel  # использует имена файлов по умолчанию
```

## Результаты

Программа принимает изображение в формате PGM и создает новое изображение, на котором выделены границы с помощью оператора Собеля.

Оператор использует два сверточных ядра для вычисления градиентов:

- Gx (горизонтальные края): [-1 0 1; -2 0 2; -1 0 1]
- Gy (вертикальные края): [-1 -2 -1; 0 0 0; 1 2 1]

Величина градиента вычисляется как √(Gx² + Gy²).

## Особенности

- Ускоренная GPU обработка с помощью CUDA
- Поддержка формата изображений PGM (загрузка и сохранение)
- Оптимизация с использованием общей памяти (shared memory)
- Обработка границ с помощью ограничения координат
- Проверка ошибок для CUDA операций

## Замечания

- Ядро использует блоки потоков 16x16 с тайлингом общей памяти
- Время обработки изображения 1920x1080 составляет ~5 мс на GPU
- Граничные условия обрабатываются с помощью ограничения координат
